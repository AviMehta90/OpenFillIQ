# Icon Placeholders Needed

For the extension to load in Chrome, you need PNG icon files in the following sizes:

- `icon-16.png` (16x16 pixels)
- `icon-32.png` (32x32 pixels)
- `icon-48.png` (48x48 pixels)
- `icon-128.png` (128x128 pixels)

## Quick Solution

Use online tools to create placeholder icons:

```bash
# Option 1: Download from placeholder.com
curl "https://via.placeholder.com/16x16/667eea/ffffff?text=F" -o icon-16.png
curl "https://via.placeholder.com/32x32/667eea/ffffff?text=F" -o icon-32.png
curl "https://via.placeholder.com/48x48/667eea/ffffff?text=F" -o icon-48.png
curl "https://via.placeholder.com/128x128/667eea/ffffff?text=F" -o icon-128.png
```

Or use any image editor to create simple colored squares.
